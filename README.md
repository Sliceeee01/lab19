# Библиотека

## Описание

Приложение для просмотра книг из личной библиотеки. Содержит список книг и детальную информацию о каждой.

## Функциональность

- Список из 5 книг
- Обложки книг
- Экран с описанием
- Навигация между экранами

## Технологии

- Kotlin
- Jetpack Compose
- Navigation Compose
- Coil
